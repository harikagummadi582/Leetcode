## 5) Code (final)

```python
class Solution:
    def groupAnagrams(self, strs: List[str]) -> List[List[str]]:
        char_map={}
        for word in strs:
            sorted_word=' '.join(sorted(word))
            if sorted_word not in char_map:
                char_map[sorted_word]=[]
                char_map[sorted_word].append(word)
            else:
                char_map[sorted_word].append(word)
        return list(char_map.values())
```
