## 1) Problem in 1 line

Check if any two meeting intervals overlap, meaning you can’t attend all meetings.

---

## 2) My first thought (before coding)

- I sorted the meetings by start time so overlaps would show up next to each other.
- Then I compared each meeting with the previously kept meeting to see if they clash.

---

## 3) Mistakes / Confusions (Error Notebook 핵심)

- Mistake #1: None (first attempt worked).
- Mistake #2: None.
- What I learned:
  - Sorting by start time turns the problem into a simple “compare with previous end” check.

---

## 4) Final approach (in plain English)

- Sort all meetings by their start time.
- Walk through them in order.
- Keep track of the last meeting you accepted.
- If the next meeting starts before the last one ends, they overlap → return `False`.
- If you finish the loop with no overlaps, return `True`.

---

## 5) Code (final)

```python
from typing import List

class Solution:
    def canAttendMeetings(self, intervals: List[List[int]]) -> bool:
        intervals.sort(key=lambda x: x[0])
        merged = []

        for interval in intervals:
            if not merged or merged[-1][1] <= interval[0]:
                merged.append(interval)
            else:
                return False

        return True
```
