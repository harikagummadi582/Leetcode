# LeetCode Patterns

This folder contains **pattern-level notes** that summarize:
- When to use a pattern
- Common mistakes
- Variations across problems

## Common patterns
- Hashing
- Two Pointers
- Sliding Window
- Binary Search
- DFS / BFS
- Dynamic Programming

## How to use this folder
- Create one markdown per pattern
- Update it whenever you solve a new problem using that pattern
- Link problems that use it

Goal:
> Recognize patterns instantly instead of starting from scratch.
