## 1) Problem in 1 line

Find the one person who is known by everyone else but does not know anyone, or return `-1` if such a person does not exist.

## 4) Final approach (in plain English)

Instead of checking every person fully, I keep one current candidate for celebrity.

I start with person `0` as the candidate and compare them with every other person.

If the current candidate knows person `i`, then the current candidate cannot be a celebrity, so I update the candidate to `i`.

If the current candidate does not know person `i`, then person `i` cannot be a celebrity, so I keep the same candidate.

After one pass, I get one possible celebrity candidate.

Then I do one final verification pass:

- the candidate should not know anyone else
- everyone else should know the candidate

If both are true, return the candidate. Otherwise return `-1`.

## 5) Code (final)

```python
# The knows API is already defined for you.
# def knows(a: int, b: int) -> bool:

class Solution:
    def findCelebrity(self, n: int) -> int:
        candidate = 0

        # Step 1: Find a possible celebrity
        for i in range(1, n):
            if knows(candidate, i):
                candidate = i

        # Step 2: Verify the candidate
        for i in range(n):
            if i == candidate:
                continue
            if knows(candidate, i) or not knows(i, candidate):
                return -1

        return candidate
```
