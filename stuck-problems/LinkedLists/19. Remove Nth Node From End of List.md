## 4) Final approach (in plain English)

- take dummy node and assign it value 0th position and head to point towards as head might want to be deleted
- assign dummy to slow and fast pointers
- take for loop for fast to do fast=fast.next for n+1 times
- while fast: move slow and fast as .next
- after while you will have slow at n+1 position from the end so do slow.next=slow.next.next
- return dummy.next as thats where the list starts

## 5) Code (final)

```python
# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, val=0, next=None):
#         self.val = val
#         self.next = next
class Solution:
    def removeNthFromEnd(self, head: Optional[ListNode], n: int) -> Optional[ListNode]:
        dummy=ListNode(0, head)
        slow=dummy
        fast=dummy
        for i in range(n+1):
            fast=fast.next
        while fast:
            fast=fast.next
            slow=slow.next
        slow.next=slow.next.next
        return dummy.next
```
