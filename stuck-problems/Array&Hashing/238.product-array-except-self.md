## 4) Final approach (in plain English)

find length of input array
take one final array with elemets as 1 multiplied length of input array
first for loop from start for prefix
first for loop - take prefix as 1, place it as first element in final array, multiple each element with prefix and place it in ith index as second step
second for loop from end for suffix
second for loop - take suffix as 1, update final array with suffix\*finalarray[i], then update suffix with multiplying suffix with current index in input array (you will figure it out with example how to write manipulations in for loops- this helped to solve it on my own)
return final array

## 5) Code (final)

```python
class Solution:
    def productExceptSelf(self, nums: List[int]) -> List[int]:
        n=len(nums)
        answer=[1]*n
        prefix=1
        for i in range(n):
            answer[i]=prefix
            prefix=prefix*nums[i]
        suffix=1
        for i in range(n-1,-1,-1):
            answer[i]=suffix*answer[i]
            suffix=suffix*nums[i]
        return answer
```
