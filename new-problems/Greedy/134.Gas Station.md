## 4) Final approach (in plain English)

- First check if the total gas is enough to cover the total cost.
- If total gas is less than cost, completing the circle is impossible.
- Move through each station and keep adding the gas difference to your tank.
- If the tank becomes negative, you cannot start from this station.
- Reset the tank to zero and set the next station as the new start.
- Continue scanning forward only once through all stations.
- If total gas is non-negative, the last start index will work.
- Otherwise, return -1.

## 5) Code (final)

```python
class Solution:
    def canCompleteCircuit(self, gas: List[int], cost: List[int]) -> int:
        tot_gain=0
        curr_gain=0
        answer=0
        for i in range(len(gas)):
            curr_gain+=gas[i]-cost[i]
            tot_gain+=gas[i]-cost[i]

            if curr_gain<0:
                answer=i+1
                curr_gain=0
        return answer if tot_gain>=0 else -1
```
