## 5) Code (final)

```python
class Solution:
    def removeDuplicates(self, nums: List[int]) -> int:
        i=0
        map={}
        while i<len(nums):
            if nums[i] not in map:
                map[nums[i]]=1
                i=i+1
            else:
                map[nums[i]]+=1
                if map[nums[i]]>2:
                    nums.remove(nums[i])
                else:
                    i=i+1
        return len(nums)
```
