## 1) Problem in 1 line

- returning array of non overlapping intervals so have to merge the end time of one interval with start time of the next interval
  Explain the problem in one sentence.

---

## 2) My first thought (before coding)

- I sorted intervals by start time so I could process them in order.
- I used start_time and end_time to track the current merged interval.
- I started looping from index 1 because the first interval initializes my merge window.
- I used end_time >= interval[0] to detect overlap.
- I used max(end_time, interval[1]) to expand the merged interval when overlaps occurred.
  What I tried first, and why.

---

## 3) Mistakes / Confusions (Error Notebook 핵심)

- Mistake #1: I appended merged intervals inside the overlap condition, which caused duplicates.
- Mistake #2: I didn’t append the final merged interval after the loop ended. I added an unnecessary i == 1 condition, which complicates the logic. I appended new intervals immediately in the else block instead of properly resetting the current merge window.
- What I learned:
- array[-1][1] accesses the last element in the second element
- I should only append a merged interval after I finish processing overlaps, not during every overlap.
- I don’t need special index conditions like i == 1; clean merge logic removes that need.
- In merge problems, I must always append the last active interval after the loop ends.
- Sorting first simplifies the logic and guarantees I only compare with the previous interval.
- Clean interval merging follows a pattern: update if overlapping, append and reset if not.

---

## 4) Final approach (in plain English)

- First, sort all intervals by their start time so overlapping ones come next to each other.
- Create an empty list called merged to store the final merged intervals.
- Go through each interval one by one.
- If merged is empty or the current interval does not overlap with the last merged interval, just add it to merged.
- If it overlaps, merge them by updating the end time of the last interval in merged to the maximum of both end times.
  Steps of the solution like you’d explain to a friend.

---

## 5) Code (final)

```python
class Solution:
    def merge(self, intervals: List[List[int]]) -> List[List[int]]:
        merged=[]
        intervals.sort(key=lambda x:x[0])
        for interval in intervals:
            if not merged or interval[0]>merged[-1][1]:
                merged.append(interval)
            else:
                merged[-1][1]=max(merged[-1][1], interval[1])
        return merged
```
