## 4) Final approach (in plain English)

- create a hashmap, take helper function
- if not node return
- add freq of node values to hashmap
- call node.left and node.right
- find max_freq
- call function
- return array of key values with highest freq in common

## 5) Code (final)

```python
# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right
class Solution:
    def findMode(self, root: Optional[TreeNode]) -> List[int]:
        freq_map={}
        def search(node):
            if not node:
                return
            if node.val not in freq_map:
                freq_map[node.val]=1
            else:
                freq_map[node.val]+=1
            search(node.left)
            search(node.right)
        search(root)
        max_freq=max(freq_map.values())
        return [k for k, v in freq_map.items() if v==max_freq]
```
