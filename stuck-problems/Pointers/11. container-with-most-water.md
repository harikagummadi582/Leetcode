## 5) Code (final)

```python
class Solution:
    def maxArea(self, height: List[int]) -> int:
        max_area=0
        width=0
        left=0
        right=len(height)-1
        while left<right:
            heightt=min(height[left], height[right])
            width=right-left
            curr_area=heightt*width
            max_area=max(max_area,curr_area)
            if height[left]<height[right]:
                left+=1
            else:
                right-=1
        return max_area
```
