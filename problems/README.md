# Problems

This folder contains all logged LeetCode problems,
organized by **problem number range**.

## Structure
problems/
├── 0001-0100/
├── 0101-0200/
├── 0201-0300/
├── ...


## Naming convention
Each problem file follows:
{problem_id}-{problem-slug}.md


Example:
0217-contains-duplicate.md

## Logging rules
- One problem = one markdown file
- Always record mistakes
- Add redo schedule before committing

Goal:
> Reach a point where problems feel familiar, not scary.
