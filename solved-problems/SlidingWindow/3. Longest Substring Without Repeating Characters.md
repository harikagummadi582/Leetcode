## 4) Final approach (in plain English)

initialise left=0, res=0, char_set
for loop for r in s
while s[r] in char_set decrease the window size by removing s[l] from charset and increment l
every iteration add s[r] to set and calculate max of res and current window=r-l+1
return res
Steps of the solution like you’d explain to a friend.

## 5) Code (final)

```python
class Solution:
    def lengthOfLongestSubstring(self, s: str) -> int:
        resLength=float('-inf')
        char_set=set()
        curr_length=0
        l=0
        r=0
        for i, char in enumerate(s):
            while char in char_set:
                char_set.remove(s[l])
                l+=1
            char_set.add(char)
            curr_length=i-l+1
            resLength=max(resLength, curr_length)
            print(curr_length, resLength)
        return resLength if resLength!=float('-inf') else 0
```

class Solution:
    def lengthOfLongestSubstring(self, s: str) -> int:
        char_set=set()
        left=0
        res=0
        for r in range(len(s)):
            while s[r] in char_set: #to avoid duplicates throughout
                char_set.remove(s[left])
                left+=1
            char_set.add(s[r])
            res=max(res, r-left+1)
        return res
