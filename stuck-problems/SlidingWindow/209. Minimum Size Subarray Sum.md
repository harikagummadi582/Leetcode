## 4) Final approach (in plain English)

- Start with two pointers and keep expanding the window by adding numbers to the running sum.
- As soon as the sum becomes ≥ target, we know we found a valid subarray.
- Record its length as a possible minimum answer.
- Then shrink from the left to see if we can make the window smaller while keeping sum ≥ target.
- Repeat this expand-then-shrink process until we finish scanning the array.
  Steps of the solution like you’d explain to a friend.

---

## 5) Code (final)

```python
class Solution:
    def minSubArrayLen(self, target: int, nums: List[int]) -> int:
        windowLength=float('inf')
        l=0
        sumWindow=0

        for i in range(0, len(nums)):
            sumWindow+=nums[i]
            while sumWindow>=target:
                windowLength=min(windowLength, i-l+1)
                sumWindow=sumWindow-nums[l]
                l+=1
            print(nums[i], sumWindow, windowLength)
        return windowLength if windowLength!=float('inf') else 0
```
