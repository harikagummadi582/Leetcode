## 1) Problem in 1 line

- Rotate a singly linked list to the right by k positions, so that the last k nodes move to the front.
  Explain the problem in one sentence.

## 4) Final approach (in plain English)

- First, walk through the list once to find its length and the last node (tail)
- Then reduce extra rotations using k = k % length (no need to rotate full cycles)
- If k becomes 0, just return the original list
- Now, connect the tail to the head → this makes the list circular
- Move to the node just before where the new head should be → (length - k - 1) steps
- The next node becomes your new head
- Break the circle by setting curr.next = None
- Return the new head

Steps of the solution like you’d explain to a friend.

## 5) Code (final)

```python
# paste final accepted code here
# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, val=0, next=None):
#         self.val = val
#         self.next = next
class Solution:
    def rotateRight(self, head: Optional[ListNode], k: int) -> Optional[ListNode]:
        if not head:
            return head
        length, tail= 1, head
        while tail.next:
            tail=tail.next
            length+=1

        k=k%length
        if k==0:
            return head
        curr=head
        for i in range (length-k-1):
            curr=curr.next
        newHead=curr.next
        curr.next=None
        tail.next=head
        return newHead

```
