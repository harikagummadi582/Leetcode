## 4) Final approach (in plain English)

- take res as array
- helper function with node and depth
- if not node return
- if len(res) equals depth then add node.val in res
- call helper function left with depth+1 and then right with depth+1
- call helper function with root and 0
- return res
  Steps of the solution like you’d explain to a friend.

## 5) Code (final)

```python
# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right
class Solution:
    def rightSideView(self, root: Optional[TreeNode]) -> List[int]:
        res=[]
        def search(node,depth):
            if not node:
                return
            if len(res)==depth:
                res.append(node.val)
            search(node.right,depth+1)
            search(node.left, depth+1)
        search(root, 0)
        return res
```
