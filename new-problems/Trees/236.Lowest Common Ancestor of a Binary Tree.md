## 1) Problem in 1 line

Find the **lowest node in a binary tree that has both nodes `p` and `q` in its subtree** (a node can also be an ancestor of itself).

---

## 2) My first thought (before coding)

I first thought about **storing parent pointers** by traversing the tree, then tracing ancestors of `p` and `q` upward until they meet. This works but requires **extra space for a parent map**, so I looked for a cleaner recursive solution.

---

## 3) Mistakes / Confusions (Error Notebook 핵심)

- I initially struggled to understand **why we check three conditions: left, right, and mid**.
- I did not realize that **LCA occurs when any two of these become true**, meaning the node connects the paths of `p` and `q`.
- I also forgot that **a node can be the ancestor of itself**, which is why `mid` is included.

**What I learned:**

- In tree recursion, **each node returns information about whether it found the target nodes in its subtree**, which helps ancestors determine the LCA.

---

## 4) Final approach (in plain English)

1. Traverse the tree using **DFS recursion**.
2. For every node check three things:
   - Did we find `p` or `q` in the **left subtree**?
   - Did we find them in the **right subtree**?
   - Is the **current node itself `p` or `q`**?

3. If **any two of these become true**, this node is the **Lowest Common Ancestor**.
4. Store that node as the answer.
5. Return whether the subtree contains `p` or `q` so the parent calls can use that information.

## 5) Code (final)

```python
# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, x):
#         self.val = x
#         self.left = None
#         self.right = None

class Solution:
    def lowestCommonAncestor(self, root: 'TreeNode', p: 'TreeNode', q: 'TreeNode') -> 'TreeNode':

        def recurse_tree(current_node: TreeNode)-> bool:

            if not current_node:
                return False

            #left recursion
            left=recurse_tree(current_node.left)

            #right recursion
            right=recurse_tree(current_node.right)

            #if current node is p or q
            mid=current_node==p or current_node==q

            if mid+left+right>=2:
                self.ans=current_node

            return mid or left or right

        recurse_tree(root)
        return self.ans
```
