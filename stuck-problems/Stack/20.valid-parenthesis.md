## 5) Code (final)

```python
class Solution:
    def isValid(self, s: str) -> bool:
        bracket_map={'(':')', '{':'}','[':']'}
        stack=[]
        for char in s:
            if char in bracket_map:
                stack.append(char)
            else:
                if not stack:
                    return False
                if bracket_map[stack.pop()]!=char:
                    return False
        if not stack:
            return True
        else:
            return False
```
