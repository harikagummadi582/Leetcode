## 4) Final approach (in plain English)

- Put the first node of every linked list into a min-heap so we can always see the smallest value.
- Repeatedly take the smallest node from the heap and attach it to the result list.
- When a node is taken from a list, put its next node into the heap.
- Keep doing this until the heap is empty and all nodes are merged into one sorted list.

````python
# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, val=0, next=None):
#         self.val = val
#         self.next = next
class HeapNode:
    def __init__(self,node):
        self.node=node
    def __lt__ (self,other):
        return self.node.val < other.node.val

class Solution:
    def mergeKLists(self, lists: List[Optional[ListNode]]) -> Optional[ListNode]:
        dummy=ListNode(0)
        current=dummy
        heap=[]

        for l in lists:
            if l:
                heapq.heappush(heap, HeapNode(l)) #push one list at once
        while heap:
            heap_node=heapq.heappop(heap) #HeapNode(1)
            node=heap_node.node #Give me the ListNode stored inside this HeapNode
            current.next=node
            current=current.next
            if node.next:
                heapq.heappush(heap,HeapNode(node.next))
        return dummy.next
````
