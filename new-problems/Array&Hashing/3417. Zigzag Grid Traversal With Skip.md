## 5) Code (final)

```python
class Solution:
    def zigzagTraversal(self, grid: List[List[int]]) -> List[int]:
        m=len(grid)
        n=len(grid[0])
        take=True
        res=[]
        for i in range(m):
            cols= range (n) if i%2==0 else range(n-1,-1,-1)
            for j in cols:
                if take:
                    res.append(grid[i][j])
                take= not take
        return res
```
