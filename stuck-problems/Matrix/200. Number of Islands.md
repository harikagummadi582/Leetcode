## 4) Final approach (in plain English)

- Go through every cell in the grid one by one.
- Whenever you see a "1" (land), that means you found a new island.
- Run DFS from that cell to mark the entire connected land (up, down, left, right) as visited by turning it into "0".
- This ensures you don’t count the same island again.
- Every time you start a new DFS, increase the island count by 1.

## 5) Code (final)

```python
class Solution:
    def numIslands(self, grid: List[List[str]]) -> int:
        rows=len(grid)
        cols=len(grid[0])
        count=0
        def dfs(r,c):
            if r<0 or r>=rows or c<0 or c>=cols or grid[r][c]=="0":
                return
            grid[r][c]="0" #mark visited
            dfs(r+1,c)
            dfs(r-1,c)
            dfs(r, c+1)
            dfs(r, c-1)
        for i in range(0, rows):
            for j in range(0, cols):
                if grid[i][j]=="1":
                    dfs(i,j)
                    count+=1
        return count
```
