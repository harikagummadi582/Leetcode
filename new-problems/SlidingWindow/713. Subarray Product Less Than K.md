## 1) Problem in 1 line

- find number of subarrays possible where the product of elements in the subarray is less than given k integer
  Explain the problem in one sentence.

## 4) Final approach (in plain English)

- Start with a “window” of numbers and keep multiplying as you move right.
- If the product becomes too big (≥ k), move the left side forward and divide out numbers until it’s small again.
- Now the window is valid, so every subarray ending at this right position works.
- How many? From l to r → that’s r - l + 1 subarrays → add them all at once.
- Repeat for every position → total count collected.
  Steps of the solution like you’d explain to a friend.

---

## 5) Code (final)

```python
class Solution:
    def numSubarrayProductLessThanK(self, nums: List[int], k: int) -> int:
        l=0
        curr_prod=1
        tot_count=0
        if k<=1: #if k is 0 or 1 no subarrays possible
           return 0
        for r, num in enumerate(nums):
            curr_prod=curr_prod*num

            while curr_prod>=k:
                curr_prod=curr_prod//nums[l]
                l+=1
            tot_count+=r-l+1
        return tot_count
```
