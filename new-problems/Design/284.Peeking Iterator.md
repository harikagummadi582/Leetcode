## 1) Problem in 1 line

- Design an iterator that can return the next element without advancing it, while still supporting normal `next()` and `hasNext()` operations. Modify a normal iterator so you can look at the next element without moving forward.

## 4) Final approach (in plain English)

- Key logic:
- peek() steals the next value once and stores it.
- next() returns stored value if peeked before.
- hasNext() checks either stored value exists OR iterator still has elements.

* Wrap the original iterator and keep a small buffer to temporarily store one element.
* Use a flag (`peeked`) to remember whether we have already looked ahead.
* When `peek()` is called, if we haven’t peeked yet, fetch the next element from the iterator and store it in `tmp`, then return it.
* If we already peeked, just return the stored value without touching the iterator.
* When `next()` is called, if we had peeked before, return the stored value and reset the flag; otherwise, just call the iterator’s `next()`.
* For `hasNext()`, return true if either we have a stored value from peeking or the underlying iterator still has elements left.

## 5) Code (final)

```python
class PeekingIterator:
     def __init__(self, iterator):
         """
         Initializes an iterator object to the beginning of a list.
         :type nums: List[int]
         """
         self.i=iterator
         self.peeked=False
         self.tmp=None

     def peek(self):
        """
        Returns the next element in the iteration without advancing the iterator.
        :rtype: int
        """
        if not self.peeked:
            self.peeked=True
            self.tmp=self.i.next() #pop next element
            return self.tmp
        else:
            return self.tmp #if already peeked just return the value

     def next(self):
         """
         Returns the next element in the iteration.
         :rtype: int
         """
         if self.peeked:
            self.peeked=False
            return self.tmp
         else:
            return self.i.next()

     def hasNext(self):
         """
         Returns true if the iteration has more elements.
         :rtype: bool
         """
         if self.peeked or self.i.hasNext():
            return True
         else:
            return False
```
