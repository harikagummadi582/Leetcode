## 5) Code (final)

```python
class Solution:
    def longestConsecutive(self, nums: List[int]) -> int:
        nums_set=set(nums)
        count=0
        longest=0
        for num in nums_set: #iterate over set to avoid duplicates
            if num-1 not in nums_set: #to start sequence with the first element
                length=1
                while num+length in nums_set: #using length to find consequtive numbers
                    length+=1
                longest=max(longest, length)
        return longest
```
