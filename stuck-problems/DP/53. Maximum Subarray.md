## 4) Final approach (in plain English)

- initialise max_sum to -inf, curr_sum to zero
- for loop and check num+curr_sum<0 then make curr_sum=0
- else add num to curr_sum
- if max_sum is still -inf then sorts nums in reverse order and return nums[0]
- return max_sum overall

Steps of the solution like you’d explain to a friend.
